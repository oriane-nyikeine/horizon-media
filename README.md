# Horizon Media — Independent Media Platform

WordPress media site covering news and editorial content.

Co-founder of the project. I handled the full technical and design side: installation, configuration, theme, structure, optimisations. My partner was in charge of the editorial direction and content management.

## What I did

- Full WordPress site setup (installation, configuration, structure)
- Child theme development based on Blogvi / Bloglo
- Three-level CSS loading chain (Bloglo → Blogvi Custom → child theme)
- Favicon optimisation for Google search results (SERP)
- Plugin configuration (SEO, performance, security)
- Ongoing maintenance and updates

## Stack

**CMS** — WordPress
**Theme chain** — Bloglo → Blogvi Custom → Child theme
**Hosting** — O2Switch
**Domain** — horizonsmedia.fr

## Child theme structure

```
horizon-media-fr-child/
├── style.css        # Child theme declaration
└── functions.php    # Three-level style loading + SERP favicon
```
